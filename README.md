# SOC Alert Investigations

This repository contains sanitized SOC alert investigation write-ups completed as part of hands-on blue team training.

The focus of these write-ups includes:
- Alert triage and investigation workflow
- Log and network traffic analysis
- Identifying indicators of compromise (IOCs)
- Determining true positives vs false positives
- Documenting conclusions and remediation recommendations

All investigations were performed in a simulated lab environment and do not contain proprietary or sensitive information.
